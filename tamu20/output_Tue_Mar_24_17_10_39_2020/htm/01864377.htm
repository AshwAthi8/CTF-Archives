Selecting the right option for you

While signing up in MSN Explorer, you will be offered a number of options. Be sure to choose the correct option for the Internet service you subscribe to. If you don't choose the correct option, you may create a duplicate MSN Internet Access account.

You don't have a connection to the Internet.

If you want to get a new MSN Internet Access e-mail address:

  • On the "Do you want MSN Internet Access?" page, select "Yes, I am interested in signing up for MSN Internet Access.

If you have an existing hotmail.com or msn.com e-mail address and want to use it as your MSN Internet Access e-mail address:

  • On the "Do you want MSN Internet Access?" page, select "Yes, upgrade my free hotmail.com/msn.com account to use MSN Internet Access."

You are an existing MSN Internet Access customer.

If you already have MSN Internet Access Member ID and connect using a modem.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on the "Do you have an MSN Internet Access dial-up account?" page, select "Yes. I have MSN Internet Access dial-up account."

If you have upgraded a free hotmail.com/msn.com account to use MSN Internet Access and connect using a modem.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on the "Do you have MSN Internet Access dial-up account?" page, select "Yes. I have MSN Internet Access dial-up account."

If you have just signed up at the store or on the Internet for MSN Internet Access and connect using a modem.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on the "Do you have MSN Internet Access  dial-up account?" page, select "Yes. I have MSN Internet Access dial-up account."

If you already have an MSN Internet Access Member ID and connect using Highspeed DSL.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on the "Do you have MSN Internet Access dial-up account?" page, select "No. I connect to the Internet using another Internet service provider, or one of the following: MSN HighSpeed Internet Access, LAN, DSL, cable modem, or satellite connection."

If you already have an MSN Internet Access Member ID and connect using Satellite.

  • On "Do you want to sign up for a new MSN Internet Access account?" page, select "No. I already have Internet access."
  • Next, on the "Do you have MSN Internet Access dial-up account?" page, select "No. I connect to the Internet using another Internet service provider, or one of the following: MSN HighSpeed Internet Access, LAN, DSL, cable modem, or satellite connection."

You have an existing non-MSN connection to the Internet and you want to keep using it.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on the "Do you have MSN Internet Access dial-up account?" page, select "No. I connect to the Internet using another Internet service provider, or one of the following: MSN HighSpeed Internet Access, LAN, DSL, cable modem, or satellite connection."

You have an existing non-MSN connection to the Internet but you want to sign up for MSN Internet Access.

If you want to get a new MSN Internet Access e-mail address:

  • On the "Do you want MSN Internet Access?" page, select "Yes, I am interested in signing up for MSN Internet Access."

If you have an existing hotmail.com or msn.com e-mail address and want to use it as your   e-mail address:

  • On the "Do you want MSN Internet Access?" page, select "Yes, upgrade my free hotmail.com/msn.com account to use MSN Internet Access."

You have DSL/LAN to connect to the Internet.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on the "Do you have MSN Internet Access dial-up account?" page, select "No. I connect to the Internet using another Internet service provider, or one of the following: MSN HighSpeed Internet Access, LAN, DSL, cable modem, or satellite connection."

You have satellite/cable modem to connect to the Internet.

  • On the "Do you want MSN Internet Access?" page, select "No. I already have Internet access."
  • Next, on "Do you have MSN Internet Access dial-up account?" page, select "No. I connect to the Internet using another Internet service provider, or one of the following: MSN HighSpeed Internet Access, LAN, DSL, cable modem, or satellite connection."
